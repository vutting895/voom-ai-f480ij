# ♊ Gemini API Integration & Configuration Guide

ไฟล์นี้กำหนดมาตรฐานความปลอดภัย ประสิทธิภาพ และการทำงานร่วมกันระหว่างระบบเซิร์ฟเวอร์ Express และ `@google/genai` SDK ในระบบ **voom AI Agent Workspace**

## 1. ⚙️ การตั้งค่าพารามิเตอร์ของโมเดล (Model Parameters)
- **Model Name:** `gemini-3.6-flash` เป็นตัวเลือกมาตรฐานเพื่อความเร็วในการพิมพ์ (Low Latency) และความสามารถในการเรียกใช้เครื่องมือที่ทรงพลัง (Excellent Function Calling)
- **Temperature:** ควบคุมระดับความคิดสร้างสรรค์ระหว่าง `0.1` ถึง `1.5` โดยมีค่าเริ่มต้นที่ `0.7`
  - *ต่ำ (0.1 - 0.3):* เหมาะกับงานคำนวณ งานสืบค้นความรู้ และงานตอบข้อเท็จจริง
  - *สูง (1.0 - 1.5):* เหมาะสำหรับงานสร้างไอเดีย แต่งนิยาย และแต่งคำกลอน
- **Max Steps:** ควบคุมความลึกสูงสุดของวงจรการใช้เครื่องมือของเอเจนต์ (สูงสุด 10 ขั้นตอน) เพื่อป้องกันปัญหา Loop ไม่รู้จบ

## 2. 🔐 ความปลอดภัยและการเก็บ Secrets
- ห้ามบันทึก API Key ในรูปแบบ Text เข้าไปในโค้ดต้นฉบับโดยเด็ดขาด
- โค้ดจะอ่าน Key จาก Environment Variable ดังนี้:
  1. `GEMINI_API_KEY` (หลัก)
  2. `GOOGLE_GENERATIVE_AI_API_KEY` (สำรอง)
- ผู้ใช้สามารถระบุ Key ได้ทันทีผ่านแท็บ **Secrets** ในแถบตั้งค่าของ AI Studio หรือ Deploy บนระบบ Cloud

## 3. 🧩 ข้อมูลของเครื่องมือที่มีให้ใช้งาน (Agent Toolset)
1. **`get_current_time`:** แสดงเวลาปัจจุบันในหน่วย UTC/เวลาท้องถิ่น
2. **`calculate_math`:** คำนวณนิพจน์ทางคณิตศาสตร์อย่างถูกต้องแม่นยำ
3. **`wikipedia_summary`:** ค้นหาประโยคสรุปข้อมูลสถิติหรือประวัติศาสตร์จากวิกิพีเดียไทย/อังกฤษ
4. **`add_to_memory`:** บันทึกข้อเท็จจริงลงคลังความจำ RAG ของผู้ใช้โดยอัตโนมัติ
5. **`call_external_api`:** เชื่อมต่อและประมวลผลข้อมูล JSON เรียลไทม์จาก Webhook / API ปลายทางที่ผู้ใช้ระบุ
