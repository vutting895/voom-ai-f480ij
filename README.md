# 🚀 voom AI Agent Workspace — Full-Stack Conversational & Build Suite

ยินดีต้อนรับสู่ **voom AI Agent Workspace** แพลตฟอร์มแชทอัจฉริยะแบบ Full-Stack ระดับโปรแกรมเมอร์ที่ยกระดับขีดความสามารถเป็นระบบ **AI Agent เต็มรูปแบบ** ขับเคลื่อนด้วย **Vercel AI SDK** และ **Express** ในตัว! มีเครื่องมือครบถ้วนตั้งแต่ระบบความทรงจำ RAG อัตโนมัติ, การคำนวณคณิตศาสตร์, ค้นหาวิกิพีเดีย, เรียกดึง API สดภายนอก และระบบจำลองการ Deploy แอปพลิเคชันด้วยคลิกเดียว

---

## 📸 รายละเอียดฟีเจอร์เด่น (Feature Highlights)

### 1. 🧠 ระบบความทรงจำอัจฉริยะ (Cognitive Memory & Auto-RAG)
- **สอนความจำ:** สั่งให้ระบบเรียนรู้ข้อมูลส่วนตัวเพียงใช้คำนำหน้าว่า `จดจำ:` (เช่น `"จดจำ: ฉันชอบดื่มกาแฟดริปอเมริกาโน่เย็นไม่หวาน"`)
- **เรียกคืนความจำ:** เอเจนต์สามารถดึงข้อมูลเหล่านั้นกลับมาตอบคำถามคุณได้ตลอดเวลาแบบไม่มีวันลืม แม้จะสลับห้องแชทหรือปิดเบราว์เซอร์ไป
- **ลบความทรงจำ:** พิมพ์สั่งการอย่างง่ายดายด้วยคำสำคัญ เช่น `ลืม:` เพื่อลบข้อมูลที่ไม่ต้องการเก็บรักษาไว้ออกจากฐานข้อมูล RAG
- **คลังความรู้:** แสดงรายการข้อมูลความทรงจำทั้งหมดบนแถบ UI **"คลังความรู้"** โดยคุณสามารถสืบค้น ค้นหา หรือกดลบความจำแต่ละรายการได้ทีละตัวอย่างอิสระ!

### 2. 🔌 แผงควบคุมเชื่อมต่อภายนอก (Custom API Webhook Integration)
- เชื่อมต่อระบบกับโลกภายนอกผ่านการตั้งค่าในแท็บ **Agent Configuration**
- สามารถกำหนด **Custom Endpoint URL** เพื่อส่งผ่านข้อมูลให้ AI ดึงผลลัพธ์มาวิเคราะห์แบบสดๆ ได้ทันที
- มีพรีเซ็ตพร้อมใช้งาน 3 รูปแบบหลัก:
  - 🪙 **Crypto Pricing (ราคาเหรียญล่าสุด):** ตรวจสอบราคา Bitcoin (BTC) และเหรียญหลักแบบเรียลไทม์ผ่าน CoinDesk API
  - 🌤️ **Live Weather (สภาพอากาศสด):** ดึงข้อมูลสภาพอากาศและอุณหภูมิปัจจุบันในกรุงเทพฯ และเมืองอื่นๆ
  - 🎭 **Jokes & Quotes (มุกตลกเบาสมอง):** ดึงข้อมูลมุกตลกเพื่อความเพลิดเพลินในการสนทนา

### 3. 🎨 แผงควบคุมความต้องการเอเจนต์ (Agent Preference Customizer Modal)
- **Model Persona Presets:** เปลี่ยนสไตล์ คาแรคเตอร์ และสุ่มคำตอบตามบทบาทต่างๆ ได้อย่างสมบูรณ์แบบ
  - *Normal:* อบอุ่น สุภาพ เป็นกันเอง
  - *Professor Voom:* เคร่งขรึม วิชาการ อธิบายอย่างลึกซึ้ง
  - *Coder Voom:* เน้นเขียนโค้ดสะอาดสไตล์แฮกเกอร์ กระชับ และเป็นเชิงเทคนิค
  - *Business Advisor:* เน้นการวางแผนกลยุทธ์เชิงธุรกิจ และแผนงานระยะยาว
  - *Custom Persona:* พิมพ์คำสั่งสั่งแต่งคาแรคเตอร์ที่คุณออกแบบเองได้อิสระ!
- **Temperature Control (ระดับความคิดสร้างสรรค์):** ปรับจูนช่วงความแม่นยำสูง (เน้นข้อเท็จจริง) ถึงระดับอิสระสูงสุด (สำหรับจินตนาการและการแต่งบทกวี)
- **Max Steps Slider:** ปรับขอบเขตความลึกสูงสุดในการคิดวิเคราะห์เครื่องมือเพื่อประมวลผลคำตอบที่ซับซ้อน

### 4. 🚀 แท็บจำลองการ Deploy อัตโนมัติ (Build & Deploy Engine)
- สั่งเก็บบิลด์แอปพลิเคชัน แปลงไฟล์ Static และจัดเตรียมแพ็กเกจขึ้น Vercel / Netlify ได้อย่างสมบูรณ์แบบ
- มีหน้าต่าง Build Console แบบเรียลไทม์ แสดงขั้นตอนการรันโปรแกรมพร้อมไฟสถานะและการเก็บสถิติลงประวัติประมวลผลงาน

## ⚡ วิธีรันเซิร์ฟเวอร์ใช้งานจริง (Local Development & Deployment)

### 1. วิธีรันในพื้นที่เครื่องคอมพิวเตอร์ของคุณ (Local Desktop Run)

เปิด Terminal ในโฟลเดอร์โปรเจกต์ แล้วทำตามขั้นตอนต่อไปนี้:

```bash
# 1. ติดตั้งไลบรารีและดีเพนเดนซีที่จำเป็นทั้งหมด
npm install

# 2. คัดลอกและสร้างไฟล์กำหนดค่า Environment Variables
cp .env.example .env

# 3. รันระบบเซิร์ฟเวอร์หลัก (Server จะเริ่มทำงานบนพอร์ต 3000 เสมอ)
npm start
```

*ตรวจสอบว่าคุณใส่คีย์ `GEMINI_API_KEY` ลงในไฟล์ `.env` เรียบร้อยแล้ว*

### 2. วิธี Deploy สู่ระบบ Cloud เปล่าฟรี 1 คลิก (Production Deploy)
- **Vercel Deployment:** คุณสามารถรันโปรเจกต์ Node.js นี้ขึ้น Vercel ได้อย่างง่ายดายผ่านหน้าต่าง Vercel CLI หรือผูกเข้ากับ GitHub Repository
- **Netlify/Cloud Run:** เหมาะสำหรับการรันเซิร์ฟเวอร์หลังบ้านแยก หรือบิลด์ไฟล์ `voom-ai.html` ขึ้น CDN ฟรีตลอดชีพ
